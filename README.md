# Commerce Components

Reusable Shopify UI components and interactive product experiences.

## Vision

Commerce Components is an open collection of reusable Shopify components designed to create premium shopping experiences.

Instead of rebuilding the same features for every store, this project provides production-ready modules that can be integrated into any Shopify theme.

---

## Current Components

### Hover Preview Panel
Interactive product preview that displays:

- Large preview image
- Product title
- Specifications
- Materials
- Reference price
- Product description

Designed for furniture, lighting, jewelry and configurable products.
